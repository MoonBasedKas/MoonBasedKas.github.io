Projects

Pizza Protocol

Pizza protocol is my first game and is a game about programming. In Pizza Protocol you're tasked with making a program which is able to navigate a city to a target destination. You are given programming blocks and must program a solution. After you've programmed your solution you then run and watch what happens. The game uses a custom VPL (visual programming language) called Pizza Bytes. A nice feature about this game is that it can become a possible IDE so long as you program the visuals, currently the game language is interpreted but you could turn this into a compiler as well if you want. I may look into further support of Pizza Byte as a language as I think a compiled VPL language is an interesting idea. This was my first expierence with Godot and the GDScript language.

K0

K0 is a kotlin toy compiler that targets x86. K0 was made using C, Bison, and FLEX. Perhaps my largest programming project yet. I worked as part of a three person team and made the compiler 4 months.

This project started after Beanweb went down and students didn't have anything to use for a semester. This was my first website and on this project I worked on the web scrapper. The webscrapper was made in Python using beatutiful soup.

Tomcat Donut Store

Tomcat Donut Store is a website made using Apache Tomcat and JSP. The website interfaces with a sql database using mysql connector. This was a five person project which was done over the course of two months. I took an active role in the development of the database and back end. In the last half of the project I would become the team lead and ensure that the team was able to complete this project. The website is meant to act as a inventory manangement and online store front for a fictional store.
